# maqui_library

This library is for the maqui sequencer by piruetas.

## Dependencies

- Boton <https://github.com/piruetasxyz/Boton>
- Perilla <https://github.com/piruetasxyz/Perilla>
- FrecuenciasMIDI <https://github.com/piruetasxyz/FrecuenciasMIDI>
- NotasMIDI <https://github.com/piruetasxyz/NotasMIDI>
- Pantalla12x8 <https://github.com/piruetasxyz/Pantalla12x8>
